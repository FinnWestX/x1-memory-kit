Frequently asked questions.

Short, honest answers about the FWX EC135 Turbine Sound Enhancement Mod and Haptic Bass Shaker Mod. If something is missing, email us at Finn.West@gmx.net.

Compatibility & Requirements

Which version of MSFS does this work with?

Microsoft Flight Simulator 2024 only. The Turbine Sound Enhancement Mod and Haptic Bass Shaker Mod were built and tested against MSFS 2024. MSFS 2020 is not supported.

Does it work with both Steam and MS Store editions?

Yes — both editions are supported. The installer auto-detects the Community folder for MS Store and Steam.

Note for Steam users: On some Steam installations the package load order needs to be adjusted inside MSFS so the FWX audio takes priority. From v1.42 onward the installer shows an explicit dialog about this step, so you can't miss it.

Do I need any extra helicopter add-on?

No. The Enhancement Mod is compatible with the free Nemeth Design EC135 that ships with MSFS 2024 — you already have it.

Important: The mod is not compatible with the paid HPG H135 or HPG H145 from Hype Performance Group. Please make sure you are flying the free Nemeth Design EC135 before purchasing.

The Haptic Bass Shaker Mod works standalone and is compatible with any 4-blade rotor helicopter in MSFS 2024, not just the EC135.

Do I need a ButtKicker or bass shaker?

The Turbine Sound Enhancement Mod works with any audio setup — headphones, speakers, or transducers. It's explicitly built to be ButtKicker-safe, meaning no unwanted bass rumble inside the cockpit audio.

The Haptic Bass Shaker Mod is designed for ButtKickers, bass shakers, or any haptic transducer. It sends dedicated rotor-pulse signals so you feel the blade rhythm physically in your chair. Without a haptic device the Haptic Engine has no effect — get the Turbine Sound Enhancement Mod only in that case.

Installation & Setup

How do I install the Turbine Sound Enhancement Mod?

Double-click the installer (.exe) you downloaded after purchase — no ZIP, no extraction. When the installer opens, choose Install. It detects your MSFS 2024 Community folder automatically. Click through the dialogs, confirm the package reorder instructions if prompted, and you're done.

Full step-by-step guide with screenshots: Installation Guide.

How do I uninstall it?

Run the same installer (.exe) you used to install — it doubles as an uninstaller. When it opens, choose Uninstall instead of Install. The Community folder entries for the Turbine Sound Enhancement Mod are removed cleanly, the default EC135 sounds come back, and the Haptic Bass Shaker Mod program is removed from your system. Each installer handles its own product; if you have the Bundle, use the Bundle installer to remove both in one go.

Does it work on Mac or Xbox?

Windows-only. MSFS 2024 is a Windows title for PC and Xbox, and the installer + Haptic Engine require Windows. Xbox cannot install third-party Community addons at all, so this is fundamentally a PC product.
