FWX EC135 Enhancement Mod — Sound

v1.44

Turbine Sound Enhancement Mod Current
April 2026

New

  • Installer now shows an explicit Package Reorder dialog. On some Steam installations the FWX package has to be loaded above the stock EC135 package — the dialog now explains this step so it can't be missed.
  • Updated product description and requirements mention the load-order note up front.

Fixes

  • Addressed the load-order case reported by some Steam users where the FWX audio was not audible after installation.

v1.41

Turbine Sound Enhancement Mod
March 2026

New

  • Full cockpit and exterior turbine audio for the EC135 in MSFS 2024.
  • 13 active engine WAVs, 20 non-engine WAVs, custom startup sequence.
  • Tuned pitch variation for a smoother, more realistic turbine response under torque changes.
  • ButtKicker-safe — no internal cockpit bass rumble.
FWX EC135 Haptic — Bass Shaker Mod

v1.51

Haptic Engine Current
April 2026

New

  • Auto-close: the Haptic Engine now exits automatically when MSFS 2024 is closed. No more background processes after you quit the sim.
  • Stability and performance refinements across SimConnect reads and device output.

v1.42

Haptic Engine
March 2026

New

  • SimConnect-based rotor impulse generation reading GENERAL_ENG_PCT_MAX_RPM:1.
  • Pause detection via ctypes — the only reliable method in MSFS 2024.
  • Autostart via exe.xml.
  • GUI with live pulse strength, rattle threshold and device dropdown.
  • Works with any 4-blade rotor helicopter in MSFS 2024 — not restricted to the EC135.
