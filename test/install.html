Installation guide.

Step-by-step setup for the FWX EC135 Sound Enhancement Pack and the FWX Rotor Haptic Engine. 5 minutes, no command line, no registry edits.

Requirements Sound Pack Haptic Engine Troubleshooting Uninstall

Requirements

  • ✓ Microsoft Flight Simulator 2024 — MS Store or Steam edition
  • ✓ Nemeth Design EC135 (for the Sound Pack — not required for the Haptic Engine)
  • ✓ Windows 10 or 11
  • ✓ ButtKicker or bass shaker (optional — only for the Haptic Engine)
  • ✓ Your Stripe order number from the confirmation email

Installing the Sound Enhancement Pack

  1. Close MSFS 2024. Make sure Microsoft Flight Simulator 2024 is fully exited before installing. If it is running, the Community folder may be locked.
  2. Download the ZIP. Open the download link from your Stripe confirmation email. You can re-download up to 3 times by entering your email and order number.
  3. Extract and run the installer. Unzip the download and double-click FWX-EC135-SoundPack-Installer.exe. Windows SmartScreen may ask for confirmation — click More info → Run anyway (the installer is not code-signed yet, it is safe).
  4. Confirm the Community folder. The installer auto-detects your MSFS 2024 Community folder for MS Store and Steam. If you use a custom location, point it to the correct folder manually.
  5. Read the Package Reorder dialog carefully. From v1.42 onward there is an explicit dialog about package load order. On some Steam installations the FWX package must be placed ABOVE the default EC135 so the new sounds take priority. Check the box to confirm you read it.
  6. Finish the install. Click through the remaining screens. No Python, no Visual C++, no drivers — just file copy.
  7. Start MSFS 2024 and load the EC135. Start a free flight with the Nemeth Design EC135. Start the engines. The new turbine should be audible immediately.
⚠️ Steam users: Package load order

If you hear the default EC135 sound after installing, the package load order inside MSFS is wrong. Inside MSFS go to Content Manager, find fwx-ec135-sound and move it ABOVE the default nemeth-design-ec135 entry. Restart MSFS.

Installing the Rotor Haptic Engine

  1. Connect your haptic device. ButtKicker, bass shaker or any transducer connected as a standard audio output device on Windows. Make sure it is recognized by Windows Sound settings before installing.
  2. Run the Haptic Engine installer. Unzip the download and run FWX-Rotor-Haptic-Engine-Installer.exe. Same SmartScreen confirmation as above.
  3. Pick your installation folder. Default is C:\Program Files\FWX\RotorHapticEngine. Accept or change it.
  4. Enable autostart. The installer offers to register the Haptic Engine in exe.xml so it starts with MSFS 2024 automatically. Leave this enabled unless you want to start it manually.
  5. Open the GUI once. After installation, launch "FWX Rotor Haptic Engine" from the Start menu. Select your haptic output device from the dropdown. Click Save.
  6. Start MSFS 2024 with any helicopter. The Haptic Engine reads rotor RPM from SimConnect and sends pulse signals to your haptic device. Works with any MSFS 2024 helicopter — not just the EC135.
✓ Tip: tuning the pulse strength

The GUI has live sliders for pulse intensity, blade rattle threshold and low-RPM behaviour. Start at default, then adjust while flying — changes apply in real time without a restart.

Troubleshooting

I still hear the default EC135 sound.

Package load order. Inside MSFS → Content Manager → move fwx-ec135-sound above nemeth-design-ec135. Restart MSFS. This is the #1 issue on Steam installations.

The Haptic Engine shows "No device found".

Windows does not see your haptic transducer as an audio output device. Open Windows Sound settings → make sure the device is listed and set as default or secondary output. Reopen the FWX GUI, it should appear in the dropdown.

SmartScreen blocks the installer.

Expected — we are not code-signed yet (code signing is €250/year, planned for a future release). Click More info → Run anyway. The installer is safe, it only writes to your Community folder and Program Files.

I bought it, where is my download?

Check your spam folder for the Stripe email with the download link. If it is not there, email Finn.West@gmx.net with your order number and we'll resend it within a few hours.

Uninstall

Both products register in Windows "Add or remove programs". Find FWX EC135 Sound Enhancement Pack or FWX Rotor Haptic Engine and click Uninstall. The Community folder entries are removed cleanly and the default sounds come back automatically.
