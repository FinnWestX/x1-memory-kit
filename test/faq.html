Frequently asked questions.

Short, honest answers about the FWX EC135 Sound Pack and Rotor Haptic Engine. If something is missing, email us at Finn.West@gmx.net.

Compatibility & Requirements

Which version of MSFS does this work with?

Microsoft Flight Simulator 2024 only. The Sound Enhancement Pack and Rotor Haptic Engine were built and tested against MSFS 2024. MSFS 2020 is not supported.

Does it work with both Steam and MS Store editions?

Yes — both editions are supported. The installer auto-detects the Community folder for MS Store and Steam.

Note for Steam users: On some Steam installations the package load order needs to be adjusted inside MSFS so the FWX audio takes priority. From v1.42 onward the installer shows an explicit dialog about this step, so you can't miss it.

Do I need any extra helicopter add-on?

No. The EC135 is already included for free with MSFS 2024 — the Enhancement Mod works with the EC135 you already have.

The Rotor Haptic Engine works standalone and is compatible with any 4-blade rotor helicopter in MSFS 2024, not just the EC135.

Do I need a ButtKicker or bass shaker?

The Sound Pack works with any audio setup — headphones, speakers, or transducers. It's explicitly built to be ButtKicker-safe, meaning no unwanted bass rumble inside the cockpit audio.

The Rotor Haptic Engine is designed for ButtKickers, bass shakers, or any haptic transducer. It sends dedicated rotor-pulse signals so you feel the blade rhythm physically in your chair. Without a haptic device the Haptic Engine has no effect — get the Sound Pack only in that case.

Installation & Setup

How do I install the Sound Pack?

Run the installer (.exe) from the ZIP you receive after purchase. It detects your MSFS 2024 Community folder automatically. Click through the dialogs, confirm the package reorder instructions if prompted, and you're done.

Full step-by-step guide with screenshots: Installation Guide.

How do I uninstall it?

Use the Windows "Add or remove programs" panel, find "FWX EC135 Sound Enhancement Pack" or "FWX Rotor Haptic Engine" and click Uninstall. The Community folder entries are removed cleanly, and the default EC135 sounds come back.

Does it work on Mac or Xbox?

Windows-only. MSFS 2024 is a Windows title for PC and Xbox, and the installer + Haptic Engine require Windows. Xbox cannot install third-party Community addons at all, so this is fundamentally a PC product.
